Narde game
=========
Narde game or Long backgammon (also known as Russian Narde)

Python implementation one of classic Backgammon game - The Long Backgammon

========

Run:   
```
git clone git@github.com:boombick/nardgame
cd nardgame
virtualenv venv
OPENROUTER_API_KEY=put_your_key_here OPENROUTER_MODEL=google/gemini-2.5-flash ./.venv/bin/python main.py
```
There is a bunch of free models on OpenRouter. You can play with LLM without extracost
